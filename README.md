## 部署步骤
### step1:下载代码

### step2:在根目录下运行安装所有依赖包
```js
   npm install 
```
### step3:在根目录下运行 webpack-dev-server 运行打包和开启服务器
###  (如果没有在全局安装webpack-dev-server的话,请全局安装此依赖包)
```js
  webpack-dev-server
```


### step4:在浏览器运行
```js
  localhost:54321 
````
